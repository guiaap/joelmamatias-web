## 🇧🇷 Português

Plataforma Web – Assessoria Documental

Plataforma em desenvolvimento para uma empresa de assessoria documental, com foco na apresentação institucional e na comercialização de conteúdos digitais.

O projeto adota o formato de Landing Page, sendo utilizado como vitrine da empresa, mas também incorpora funcionalidades adicionais, como:

- Blog institucional
- Venda de produtos digitais (e-books, materiais informativos, etc.)

Até o momento, o projeto está sendo desenvolvido com as seguintes tecnologias:

- React
- Tailwind CSS

Objetivos do projeto:

- Criar uma plataforma moderna, responsiva e de fácil manutenção, que permita:
- Divulgar os serviços da empresa
- Publicar conteúdos informativos
- Oferecer produtos digitais de forma integrada

## 🇺🇸 English

Web Platform – Document Management Consultancy

Platform under development for a document management consultancy company, focused on institutional presentation and digital product sales.

The project follows a Landing Page style, acting as the company’s showcase, while also including additional features such as:

- Institutional blog
- Digital product sales (e-books, informational materials, etc.)

So far, the project is being developed using the following technologies:

- React
- Tailwind CSS

Project goals:

- To build a modern, responsive, and easy-to-maintain platform that allows:
- Promoting the company’s services
- Publishing informational content
- Offering digital products in an integrated way
